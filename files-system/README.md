
### File Manipulation Commands
- `touch` - Create a new empty file
  ```bash
  touch filename.txt
  ```

- `mkdir` - Create a new directory
  ```bash
  mkdir dirname
  ```

- `rm` - Delete a file
  ```bash
  rm filename.txt
  ```

- `rmdir` - Delete an empty directory
  ```bash
  rmdir dirname
  ```

- `rm` - Remove non-empty directories
  ```bash
  rm -r dirname
  ```
